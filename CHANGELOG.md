# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3] - 2025-12-23

### Changed
- Added GitHub repository URL to package.json
- Added homepage and bug tracker links
- Improved npm package metadata

## [1.0.2] - 2025-12-23

### Changed
- Removed Effect.js related content from README
- Improved documentation clarity

## [1.0.1] - 2025-12-23

### Fixed
- Resolved React 19 version compatibility issues
- Updated peer dependencies to support both React 18 and 19

## [1.0.0] - 2025-12-23

### Added
- Initial release
- Type-safe EventService for in-app communication
- BroadcastService for cross-window/tab communication
- React hooks: useEventEmit, useEventOn, useEventState
- React hooks: useBroadcast, useBroadcastOn
- Full TypeScript support
- Zero dependencies (React as peer dependency)
- Comprehensive documentation
